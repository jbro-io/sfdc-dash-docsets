DcSocialProfile

The DcSocialProfile object is a read-only object accessible only through the Data.com Social Key API. This object is available in API version 32.0 or later.

Supported Calls

describeSObjects(), query()

Special Access Rules

This is a read-only object that is accessed by only the Data.com Social Key API
