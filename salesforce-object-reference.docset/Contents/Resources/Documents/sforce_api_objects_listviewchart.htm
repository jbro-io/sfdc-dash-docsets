ListViewChart

Represents a graphical chart that’s displayed on Salesforce1 list views. The chart aggregates data that is filtered based on the list view that’s currently displayed. This object is available in API version 33.0 and later.

Supported Calls

create(), delete(), describeSObjects(), query(), retrieve(), update(), upsert()
