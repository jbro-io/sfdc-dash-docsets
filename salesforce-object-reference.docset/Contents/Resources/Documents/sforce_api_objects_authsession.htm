AuthSession

The AuthSession object represents an individual user session in your organization. This object is available in versions 29.0 and later.

Usage

The AuthSession object exposes session data and enables read and delete operations on that data. For example, use this object to create a report showing who is signed in to your organization, or to create a tool to delete a session, ending that userâ€™s session. For a user, only their own sessions are available, while administrators can see all sessions.
