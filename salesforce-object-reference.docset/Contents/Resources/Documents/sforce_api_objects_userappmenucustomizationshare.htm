UserAppMenuCustomizationShare

Represents a sharing entry on a UserAppMenuCustomization record. This object is available in API version 35.0 and later.

Supported Calls

create(), delete(), describeSObjects(), query(), retrieve(), update(), upsert()

Fields

Field Name Details

A value that represents the type of sharing allowed. The possible values are:
  • Read
  • Edit

Properties
Create, Filter, Group, Sort

RowCause
Type
picklist
Properties
Create, Filter, Group, Nillable, Restricted picklist, Sort
Description
Reason that this sharing entry exists.
Values may include:
  • Manual—The User or Group has access because a user with “All” access manually shared the UserAppMenuCustomization record with them.
  • Owner—The User is the owner of the UserAppMenuCustomization record or is in a role above the UserAppMenuCustomization record owner in the role hierarchy.
