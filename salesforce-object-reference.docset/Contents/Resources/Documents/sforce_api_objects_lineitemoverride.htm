LineitemOverride

A forecast override of a line item on an Opportunity. This read-only object for customizable forecasting has a child-parent relationship with OpportunityOverride.

Supported Calls

create(), describeSObjects(), getDeleted(), getUpdated(), query(), retrieve(), update(), upsert()

Special Access Rules

This object is only accessible if your organization has enabled the customizable forecasting feature, which can be done in the user interface. Requires the “View All Data” permission.
