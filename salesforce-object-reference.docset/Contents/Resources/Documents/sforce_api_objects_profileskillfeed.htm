ProfileSkillFeed

Represents a single feed item in the feed displayed on the detail page for a profile skill.

A ProfileSkill feed shows recent changes to a ProfileSkill record for any fields that are tracked in feeds, and comments and posts about the record. It is a useful way to stay up-to-date with changes to solutions.

Supported Calls

delete(), describeSObjects(), getDeleted(), getUpdated(), query(), retrieve()
