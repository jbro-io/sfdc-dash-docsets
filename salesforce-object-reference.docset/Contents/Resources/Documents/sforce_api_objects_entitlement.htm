Entitlement

Represents the customer support an Account or Contact is eligible to receive. This object is available in API version 18.0 and later. Entitlements may be based on an Asset, Product2, or ServiceContract.

Supported Calls

create(), delete(), describeLayout(), getDeleted(), getUpdated(), query(), retrieve(), search(), undelete(), update(), upsert()
