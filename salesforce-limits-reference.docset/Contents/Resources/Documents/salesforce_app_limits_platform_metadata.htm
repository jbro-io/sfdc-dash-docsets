Metadata Limits

The following limits apply to the Force.com IDE, the Force.com Migration Tool, and the Metadata API.

Limit Description
Retrieving and deploying metadata Metadata API can deploy and retrieve up to 10,000 files or 400 MB (39 MB compressed) at one time. If any of these limits is exceeded, the deployment or retrieval fails.
Change sets Inbound and outbound change sets can have up to 10,000 files of metadata.
