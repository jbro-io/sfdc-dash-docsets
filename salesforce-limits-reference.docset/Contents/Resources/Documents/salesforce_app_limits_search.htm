Search Limits

The following file extensions and maximum file sizes are supported for Documents search.

File Type File Extensions Maximum Size
HTML (only the text within a <body> tag) .htm, .html, .xhtml 5 MB
PDF .pdf 25 MB
PPT .pot, .pps, .ppt 25 MB
RTF .rtf 5 MB
Text .c, .cpp, .css.csv, .ini, .java, .log, .sql, .txt 5 MB
Word .doc, .dot 25 MB
XLS .xls, .xlt 5 MB, or a maximum cell limit of 100,000 cells
XML .xml 5 MB

Note

The contents of documents that exceed the maximum sizes are not searched; however, the document fields are still searched. Only the first 1,000,000 characters of text are searched. Text beyond this limit is not included in the search.
